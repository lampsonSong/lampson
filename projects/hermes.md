# hermes

- **路径**: `~/.hermes/hermes-agent`
- **类型**: 自改进 AI Agent（self-improving AI agent）
- **语言**: Python >= 3.11
- **版本**: 0.10.0
- **来源**: Nous Research（MIT License）

## 主要结构

| 目录/文件 | 说明 |
|----------|------|
| agent/ | Agent 核心实现 |
| hermes_cli/ | CLI 命令行工具 |
| skills/ | 技能模块 |
| tools/ | 工具集 |
| environments/ | 环境配置 |
| plugins/ | 插件 |
| gateway/ | 网关服务 |
| tui_gateway/ | TUI 网关 |
| acp_adapter/ | ACP 适配器 |
| cli.py | CLI 主入口 |
