# Exploration Instinct

探索能力不是需要调用的工具，而是 agent 的内在行为准则。

## 核心原则

1. **不猜** — 不确定的事情先探测（ls、cat、which、env），不要猜测
2. **先看再做** — 动手前先了解现状，不要上来就改
3. **追问到底** — 遇到未知不要跳过，探索到理解为止
4. **记录发现** — 探索中发现的新知识要及时沉淀（projects/、skills/）

## 探索方法

- 路径不明 → `which` → `head` → `ls` 逐步定位
- 行为不明 → 跑一下看输出、看日志
- 依赖不明 → `pip show`、`import`、`--help`
- 配置不明 → 读配置文件、环境变量
